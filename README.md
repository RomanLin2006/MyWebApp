# Веб-приложение для визуального мониторинга статуса алкогольных лицензий предприятий Москвы

## Описание проекта

Веб-приложение для визуального мониторинга статуса алкогольных лицензий предприятий Москвы на основе открытых данных Правительства Москвы.

## Функционал

- Интерактивная карта Москвы с маркерами предприятий
- Цветовая индикация статуса лицензий (зелёный/жёлтый/красный)
- Детальная информация о предприятиях
- Система фильтров и поиска
- k-means кластеризация для анализа плотности
- Зоны риска с цветовой разметкой
- Расширенный поиск с автодополнением
- Умные подсказки и анализ
- Информационные слои карты

## Технологии

- **Backend**: Python, Flask
- **Frontend**: HTML, CSS, JavaScript, Leaflet.js
- **База данных**: MySQL
- **Машинное обучение**: k-means

## Установка и запуск

### Требования

- Python 3.9+
- MySQL 8.0+
- Node.js и npm (для frontend зависимостей)

### Установка зависимостей

```bash
# Backend
cd backend
pip install -r requirements.txt

# Frontend
cd frontend
npm install
```

### Настройка базы данных

1. Создайте базу данных MySQL
2. Импортируйте структуру и процедуру из папки `database`
3. Заполните данными из датасета

### Запуск приложения

```bash
# Backend
cd backend
python app.py

# Frontend
# Откройте frontend/index.html в браузере
# Или используйте Live Server в VS Code
```

## Структура проекта
